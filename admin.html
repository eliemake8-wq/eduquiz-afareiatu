📌 Comment déployer un QCM ?
  1. Crée et sauvegarde un QCM dans l'éditeur
  2. Clique ⬇️ "Exporter JSON" → copie le fichier dans data/
  3. Ajoute le QCM dans index.html dans le tableau CATALOGUE
  4. Redéploie : netlify deploy --prod --dir=.
